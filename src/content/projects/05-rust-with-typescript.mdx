## Problem

대부분의 Rust 학습 자료가 C/C++ 배경을 전제합니다. TypeScript 개발자가 Rust를 배울 때 쓸 수 있는 멘탈 모델 브리지가 없었습니다.

## Approach

**Astro + Starlight**로 인터랙티브 학습 가이드를 만들었습니다. 커스텀 Astro 플러그인으로 코드 블록에서 Cloudflare Workers의 **Rust/Axum 백엔드**에 요청을 보내 **브라우저에서 직접 Rust 코드를 실행**할 수 있게 했습니다. `Accept-Language` 미들웨어로 KO/EN 자동 라우팅을 구현하고, 모든 챕터에 TypeScript ↔ Rust 병렬 예제를 넣었습니다.

## Result

Cloudflare Pages 라이브 배포. 문법·소유권·트레이트·동시성 등 **14개 챕터** 완성. 브라우저 내 Rust 코드 실행. 한국어/영어 양쪽 완성. C/C++ 배경 없이도 읽을 수 있는 유일한 TypeScript 개발자용 Rust 가이드.
